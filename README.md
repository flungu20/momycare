# MomyCare

**Care for your child. Care for you.**

MomyCare este o aplicație cozy, calmă, pentru părinți (mame și tați), focusată pe:
- **Copil**: activități pe vârstă (0–3, 3–7, 7–12, 12–20)
- **Părinte**: suport emoțional (chat)
- **Fără diagnostic medical** (doar sprijin general)

---

## Stack tehnic

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Stocare: `localStorage`
- AI: **DEMO MODE** (răspuns fix, fără billing OpenAI)
